CloudJS - A Simple Javascript Library Which help your blog or personal site's visitor to save your file directly to their Cloud Storages (DropBox, OneDrive, Google Drive etc.)

Welcome to CloudJS.

CloudJS is a simple jQuery plugin, Let your visitor saves a file into their DropBox, Google Drive etc

Say Good Bye!!! to "Save As Computer" file sharing system.

CloudJS specially build to serve file sharing most efficiently and securely over www with the help of popular Cloud Service Providers. e.g. DropBox, GoogleDrive more coming soon.

Best Fit

  1. Blogger
  2. Personal
  3. Magazine
  4. Document Seller
  5. And Any number of services which related to file sharing over websites or application.
